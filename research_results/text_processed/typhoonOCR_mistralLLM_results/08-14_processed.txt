To display these routes on a map, we can create a custom React component for the temple complex map, where each route is represented as a line connecting two points. Here's an outline of how I would implement it using the MapboxGL library:

1. First, install MapboxGL in your project using npm or yarn (`npm install @mapbox/mapbox-gl`)
2. Create a new file for the TempleComplexMap component and import necessary dependencies, including React, MapboxGL, and the routes data from the parent component:

```javascript
import React, { useState } from 'react';
import mapboxgl from '@mapbox/mapbox-gl';
import routesData from './routesData';

mapboxgl.accessToken = yourMapboxAccessToken;
```

3. Create a new component called TempleComplexMap:

```javascript
const TempleComplexMap = ({ center, zoom }) => {
  // ...
};
```

4. In the TempleComplexMap component, initialize the map using MapboxGL and set up event listeners for when the component is mounted and unmounted:

```javascript
const TempleComplexMap = ({ center, zoom }) => {
  const [map, setMap] = useState(null);

  useEffect(() => {
    // Initialize map
  }, []);

  return (
    <div style={{ height: '100%' }}>

    </div>

```

5. Inside the useEffect, create and initialize the map when the component is mounted. Add the necessary layers, such as a tile layer for the base map and a layer for the routes:

```javascript
const TempleComplexMap = ({ center, zoom }) => {
  const [map, setMap] = useState(null);

  useEffect(() => {
    if (!map) {
      const map = new mapboxgl.Map({
        container: document.querySelector('.temple-complex-map'),
        style: 'mapbox://styles/mapbox/streets-v11',
        center,
        zoom,
      });

      // Add a layer for the routes
      map.on('load', () => {
        map.addLayer({
          id: 'routes',
          type: 'line',
          source: {
            type: 'geojson',
            data: routesData,
          },
          layout: {
            'line-join': 'round',
            'line-cap': 'round',
          },
          paint: {
            'line-color': ['get', 'color'],
            'line-width': 4,
          },
        });
      });
    }

    return () => setMap(null); // Clean up on unmount
  }, []);

  return (
    <div style={{ height: '100%' }}>
      <div className="temple-complex-map" />
    </div>
  );
};
```

This implementation creates a new MapboxGL map, adds the provided routesData as a GeoJSON source for lines, and sets the line color and width using the data's 'color' and 'width' properties.
