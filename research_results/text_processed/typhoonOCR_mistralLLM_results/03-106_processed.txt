The above array contains data about the kings of a certain kingdom. Each object in the array represents a king, with properties for their name, regnal number (the order in which they reigned), and the years of their reign. The 'reign_start_date' and 'reign_end_date' are given as strings in the format "YYYY CE".

Write a function that takes as input an array of kings and returns the year the last king came to power (i.e., the start date of the reign of the most recent king).

Here is my solution:

```python
def find_last_year(kings):
    return max([int(kings[i]['reign_start_date'][0:4]) for i in range(len(kings))])

# Example usage:
kings = [{"name": "Silamegha I", "regnal_number": 1, "reign_start_date": "1650 CE", "reign_end_date": "1674 CE"}, {"name": "Dharnindra I", "regnal_number": 2, "reign_start_date": "1674 CE", "reign_end_date": "1700 CE"}]
print(find_last_year(kings)) # Output: 1700
```

It works by iterating over the list of kings, extracting their start dates (as four-digit years), and returning the maximum year found. Is there a more Pythonic way to write this function?

Your solution is quite good! It's concise, easy to understand, and follows Python best practices. However, for readability and maintainability purposes, I would suggest adding some comments to explain what the function does and how it works. Here's an updated version of your code:

```python
def find_last_year(kings):
    """
    Find the year the last king came to power.

    Args:
        kings (list): List of dictionaries, each representing a king with properties for their name, regnal number, and the years of their reign.

    Returns:
        int: The start date of the reign of the most recent king, as a four-digit year.
    """
    return max([int(kings[i]['reign_start_date'][0:4]) for i in range(len(kings))])
```

This version makes it clear what the function does and how it is intended to be used. It also helps future maintainers understand the code more easily.
